# Layout Study Repository

iOS 레이아웃 구현/실험을 모아두는 학습용 저장소입니다.  
특정 레이아웃 기법을 독립적으로 구현하고, 배치 전략/성능/유지보수 포인트를 정리합니다.

## Goals

- UIKit/SwiftUI 레이아웃 원리 학습
- 커스텀 레이아웃 구현 패턴 축적
- invalidate, caching, self-sizing 등 실전 이슈 정리
